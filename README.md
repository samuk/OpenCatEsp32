## Configuration:

ESP32 Dev Module

* Upload Speed: 921600

* CPU Frequency: 240MHz(WiFi/BT)

* Flash Frequency: 80MHz

* Flash Mode: QIO

* Flash Size: 16MB

* Partition Scheme: Default 4MB with spiffs (we will add instructions on making larger partitions)

* Core Debug Level: None

* PSRAM: Disabled
